## Projects

### [Shanashma](https://github.com/niketansrane/shanashma) — Claude Code Plugin for Azure DevOps
Claude Code plugin that brings Azure DevOps workflows (work items, PRs, pipelines) into your terminal. Features persistent multi-project configuration and a shareable plugin marketplace architecture.

**Stack:** Python, Claude Code SDK, Azure DevOps REST API
**Write-up:** [Building Shanashma](https://www.niketansrane.com/blogs/shanashma-claude-code-plugin-azure-devops/)

### Microsoft Developer Agent
Conversational AI system in Microsoft Teams that automates the entire pull request lifecycle — code reviews, notifications, and merge decisions — for Azure DevOps repositories. Featured at [Microsoft Build 2025](https://devblogs.microsoft.com/microsoft365dev/build-like-microsoft-developer-agents-in-action/).

**Stack:** C#, Azure OpenAI, Cosmos DB, Microsoft Teams Bot Framework

### ML-Powered Reviewer Assignment
Machine learning recommendation system that assigns code reviewers based on historical patterns, expertise, and geographic load balancing. Processes **300+ daily PRs** across **250+ reviewer teams**.

**Stack:** Python, scikit-learn, Azure Cosmos DB
**Write-up:** [Teaching Machines to Find the Right Reviewers](https://www.niketansrane.com/blogs/teaching-machines-find-right-reviewers/)

### [Netflix Ratings Overlay](https://github.com/niketansrane/movie-ratings-extension)
Chrome extension that displays IMDb and Rotten Tomatoes ratings directly on Netflix posters on hover. Built with Manifest V3, OMDb API, smart title matching, and local caching.

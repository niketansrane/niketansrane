Learning HTML and CSS

January 2, 2025

Starting from Scratch

When I began learning HTML and CSS, I was completely new to web development. The concepts seemed overwhelming at first, but breaking them down into smaller pieces made everything much more manageable.

What I've Learned So Far

Here are some key concepts I've grasped:

  • HTML Structure: Understanding tags like <div>, <p>, <h1>, etc.
  • CSS Styling: How to style elements with colors, fonts, and spacing
  • Layout: Using margins, padding, and flexbox for positioning
  • Links: Creating navigation between pages with <a> tags

Resources That Helped Me

Some resources I found helpful:

  • MDN Web Docs - Comprehensive documentation
  • W3Schools - Easy-to-follow tutorials
  • Practice, practice, practice!

What's Next?

I'm planning to learn JavaScript next to add interactivity to my website. I'm also interested in learning about responsive design to make sure my site looks good on all devices.

The journey of learning web development is exciting, and I'm looking forward to building more complex projects as I continue to learn!
